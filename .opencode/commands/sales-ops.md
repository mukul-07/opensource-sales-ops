sales-ops router. Arguments provided: "$ARGUMENTS"

If arguments contain a prospect target (URL, LinkedIn company page, domain, or a short company name), the skill will execute auto-pipeline mode (qualify + report + tracker).

Otherwise, the discovery menu will be shown.

Load the sales-ops skill:
```
skill({ name: "sales-ops" })
```
